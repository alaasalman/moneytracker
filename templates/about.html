MoneyTracker is a personal finance application that tries to help you in answering the seemingly simple questions above.

The idea of this project occurred to me after I started wondering how I spend my money. So I started to record payments in a spreadsheet and then turned to using wxbanker which is a simple personal finance desktop application. But then as my needs changed, I decided to write my own.

What powers this site:

  • Django
  • Gunicorn
  • Nginx
  • Postgresql
  • Highcharts
